# 媒体

属性名:`@media (运算符) [type] (character)`

媒体类型:

| 值      | 含义   |
| ------ | ---- |
| all    | 所有   |
| screen | 电子屏幕 |
| print  | 打印机  |

媒体特性:

| 值                | 含义                                            |
| ---------------- | --------------------------------------------- |
| width            | 视口宽度                                          |
| max-width        |                                               |
| min-width        |                                               |
| height           | 视口高度                                          |
| max-height       |                                               |
| min-height       |                                               |
| device-width     |                                               |
| max-device-width |                                               |
| min-device-width |                                               |
| orientation      | 视口旋转方向<br/>1. portrait 纵向<br/>2. landscape 横向 |

运算符:

| 值     | 含义  |
| ----- | --- |
| and   |     |

| not   |     |
